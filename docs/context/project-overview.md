# Project Overview

## Proizvod

Montenegro Coast City Guide je premium mobilni travel frontend za otkrivanje mjesta na crnogorskoj obali. Fokus je na brzom pronalasku kvalitetnih lokalnih preporuka uz image-led UI, dobru pretragu i jednostavno čuvanje favorita.

## Trenutni opseg

- Frontend only
- Expo React Native aplikacija u root folderu
- Postojeći backend ostaje izolovan u `backend/`
- Mock i real API režim podržani iz iste UI arhitekture

## Ključne korisničke potrebe

- Brz izbor opštine i kategorije
- Pouzdana globalna pretraga
- Detalji mjesta sa dovoljno konteksta za odluku
- Lokalno čuvanje favorita
- Otvaranje rute u nativnim mapama bez komplikovanja ugrađenim mapama
